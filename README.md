### Decodificador simples que converte todas as vogais de um texto em binário.

Foi utilizado a seguinte lógica de criptografia:
<br/>
A -> 01100001
</br>
E -> 01100101
</br>
I -> 01101001
</br>
O -> 01101111
</br>
U -> 01110101
</div>
